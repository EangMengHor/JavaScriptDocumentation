Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language, every browser has a JavaScript interpreter(used to understand JavaScript's codes).

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elemetns such as operators, control, structures, and statements. Core JavaScript can be exteneded for a variety of purposes by supplementing it with additional objects; for instance:

  • Client-side (someone who uses your website) JavaScript extends the core language by supplying objects to control a browser and its Document Objct Model (MOD). For example, Client-side extenstions allow an application to place elements on an HTML form and respond to the user when a button is clicked, mouse clicks, form input, and page navigation.
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in some others. The JavaScript langauge resembles Java but does not have Java's static typing and strong type checking. JavaScript follows most Java expression syntax, naming conventions and basic control-flow constructs which was the reason why it was renamed from LiveScript to JavaScript.

Some of the main differences are:

  • Java code must be compiled, and JavaScript code is all-text.
  • JavaScript code is run on a browser only, while Java creates applications that run in a virtual machine or browser.
  • Java is an OOP (object-oriented programming) language, and JavaScript is specifically an OOP scripting language.

Variables

Variables in JavaScript are like those of Math. They are used to store values in your application. the names of variables, called identifiers, conform to certain rule.

A JavaScript identifiers must start with a letter, underscore(_), or dollar sign ($); the next characters can also be digits (0-9). And JavaScript is case sensitive meaning a variable called x is different from X.

Declaring variables
You can declare a variable in 3 ways

With the keyword var. For example, var x = 42 This syntax can be used to declare both local and global variables.

By simply assigning it a value, this is the easiest way. For example, x = 42 Even though, it is the easist it is not the safest. This always declares a global variable. It generate a strict JavaScript warning.

With the keyword let. For example, let y = 12 This syntax can be used to declare a block scope local variable, See Variable Scope below.

Variable Scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the current document. When you declare a variable within a function it is called a local variable, because it is only available within the function defined.

if (true) { let y = 5} console.log(y)

This would not print out 5, this is because y is only defined inside the if statement(local variable)

Global Variable

Global variables are in fact properties of the global object. In web pages the global object is window, so you can set and access global variables using the window.variable syntax.

Consequently, you can access global variables declared in one window or frame from another window or frame by specifying the window or frame name. For example, if a variable called phoneNumber is declared in a document, you can refer to this variable from an iframe as parent.phoneNumber.

Constants

You can create a read-only, name constant with the const keyword. The syntax of a constant identifier is the same as for a variable identifier. It must start with a letter, underscore(_), or a dollar sign ($).

const PI = 3.14

A constant cannot change value through the document or be re-declared while the script is running.

The scope rules for constants are the same as those for let block scope variables. You cannot declare a constant with the same name as a function or variable in the same scope. For example,

function f() {}; const f = 5; // This will cause an error. function f() {const g = 5, var g;}; // This will also cause an error.

if... else statement
Use the if statement to execute a statement if a logical condition is true. Use the optional else clause to execute a statement when the condition is false. An if statement looks as follow: if (condition){ statement_1; } else { statement_2;} Condition can be any expression that evaluates to true or false. For example, (1>2) would evaluate to false. If the first statement is executed (true) then it stops, else if it is false it goes to the second statement. Having multiple conditions is fine, for example: if (condition_1){ statement_1; } else if (condition_2){ statement_2; }else{ statement_3 }
while statement
A while statement exrcutes its as long as a specified condition evaluates to true. A while statement looks as follows: while (condition) statement If the condition becomes false, statemnet within the loop stops.

The condition test occurs before statement in the loop is executed. If the condition returns true, statement is executed and the condition is tested again. If the condition returns false, execution stops and control is passed to the statement following while.

TO execute multiple statements, use a block statement ({...}) to group those statements

Example:

The following while loop iterates as long as n is less than three:

var n = 0; var x = 0; while (n < 3) {n++; x+= n;

With each iteration, the loop increments n and adds that value to x. Therefore, x and n take on the following values:

  • After the first pass: n = 1 and x = 1
  • After the second pass: n = 2 and x = 3
  • After the third pass: n = 3 and x = 6

After completing the third pass, the condition n < 3 is no longer true, so the loop terminates.

Function declarations
A function defintion (also called a function declaration, or function statement) consists of the function keyword, followed by:
  • The name of the function
  • A list of arguments to the function, enclosed in parentheses() and seperated by commas(;).
  • The JavaScript statement that define the function, enclosed in curly brackets({}).

For example, the following code defines a simple function named squareNun:

function squareNum(number) {return number * number}

The function square takes on argument, call number. The function consists of one statement that says to return the arguement(number) of the function, multiplied by itself.
